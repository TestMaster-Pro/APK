How to Use

📚 Selecting an Exam

Tap on any exam pill (like SSC, UPSC, JEE) above the test list. This filters all available tests for that exam. Tap again to deselect.

🔍 Filtering Tests

After selecting an exam, use the Subject, Chapter dropdowns and Search bar to narrow down tests. Filters update instantly.

▶️ Starting a Test

Click the Start Test button on any test card. This opens the test interface in a new page where you can answer questions with a timer.

📊 Viewing Results

Switch to the My Results tab to see your test history grouped by exam. Use Open Full Results for detailed analytics.

🔄 Retaking Tests

Already attempted tests show a Retake button and a green score preview with your best attempt details.

👤 Profile & Settings

Open the Profile page from the sidebar to update your name, mobile, email, state, education, and change your password. You can also set a profile picture.

💬 Chat Feature

Use the Chat button in the navbar to get instant help or discuss questions with the community.

🔔 Notices

The notice bar shows the latest announcement. Click All notices to view the full list of updates and alerts.

📈 Stats Cards

The 4 stat cards at the top show: Available tests, Completed tests, Average Score, and Best Score for the selected exam.
